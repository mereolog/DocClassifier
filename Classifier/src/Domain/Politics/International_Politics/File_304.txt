href="/wiki/Umowa_mi%C4%99dzynarodowa" title="Umowa międzynarodowa">umowy międzynarodowe</a> zawarte między państwami europejskimi, które dały podwaliny pod organizację i porządek prawny Unii. Z punktu widzenia <a href="/wiki/Prawo_mi%C4%99dzynarodowe_publiczne" title="Prawo międzynarodowe publiczne" class="mw-redirect">prawa międzynarodowego publicznego</a> są to umowy międzynarodowe, jednak w stosunkach wewnętrznych (między państwami członkowskimi) ustanowiły one autonomiczny porządek prawny i są obecnie elementem <a href="/wiki/Prawo_Unii_Europejskiej#Prawo_pierwotne" title="Prawo Unii Europejskiej">prawa pierwotnego Unii Europejskiej</a>.</p><table id="toc" class="toc"><tr><td><div id="toctitle"><h2>Spis treści</h2></div><ul><li class="toclevel-1 tocsection-1"><a href="#Traktaty"><span class="tocnumber">1</span> <span class="toctext">Traktaty</span></a><ul><li class="toclevel-2 tocsection-2"><a href="#Traktaty_akcesyjne"><span class="tocnumber">1.1</span> <span class="toctext">Traktaty akcesyjne</span></a></li></ul></li><li class="toclevel-1 tocsection-3"><a href="#Nieratyfikowane"><span class="tocnumber">2</span> <span class="toctext">Nieratyfikowane</span></a></li><li class="toclevel-1 tocsection-4"><a href="#Inne_zwi.C4.85zane_z_UE"><span class="tocnumber">3</span> <span class="toctext">Inne związane z UE</span></a></li><li class="toclevel-1 tocsection-5"><a href="#Zobacz_te.C5.BC"><span class="tocnumber">4</span> <span class="toctext">Zobacz też</span></a></li><li class="toclevel-1"><a href="#Przypisy"><span class="tocnumber">5</span> <span class="toctext">Przypisy</span></a></li><li class="toclevel-1 tocsection-6"><a href="#Linki_zewn.C4.99trzne"><span class="tocnumber">6</span> <span class="toctext">Linki zewnętrzne</span></a></li></ul></td></tr></table><h2><span class="editsection">[<a href="/w/index.php?title=Traktaty_Unii_Europejskiej&amp;action=edit&amp;section=1" title="Edytuj sekcję: Traktaty">edytuj</a>]</span> <span class="mw-headline" id="Traktaty">Traktaty</span></h2><ul><li><i><b><a href="/wiki/Traktat_paryski_(1951)" title="Traktat paryski (1951)">Traktat ustanawiający Europejską Wspólnotę Węgla i Stali</a></b></i> - podpisany w <a href="/wiki/Pary%C5%BC" title="Paryż">Paryżu</a> 18 kwietnia <a href="/wiki/1951" title="1951">1951</a>, wszedł życie 23 lipca <a href="/wiki/1952" title="1952">1952</a>. Był on rezultatem pomysłu i debaty nad powołaniem organizacji regulującej przemysł wydobywczy i stalowy. Traktat ten, zwany <i>Paryskim</i> powołał do życia <a href="/wiki/Europejska_Wsp%C3%B3lnota_W%C4%99gla_i_Stali" title="Europejska Wspólnota Węgla i Stali">Europejską Wspólnotę Węgla i Stali</a>. Układ zawarło sześć państw - <a href="/wiki/Francja" title="Francja">Francja</a>, <a href="/wiki/Niemcy" title="Niemcy">RFN</a>, <a href="/wiki/W%C5%82ochy" title="Włochy">Włochy</a>, <a href="/wiki/Belgia" title="Belgia">Belgia</a>, <a href="/wiki/Holandia" title="Holandia">Holandia</a> i <a href="/wiki/Luksemburg" title="Luksemburg">Luksemburg</a>. Obowiązywał on 50 lat, w <a href="/wiki/2002" title="2002">2002</a> roku kompetencje EWWiS przejęła <a href="/wiki/Wsp%C3%B3lnota_Europejska" title="Wspólnota Europejska">Wspólnota Europejska</a>.</li><li><i><b><a href="/wiki/Traktaty_rzymskie" title="Traktaty rzymskie">Traktat ustanawiający Europejską Wspólnotę Gospodarczą i Traktat ustanawiający Europejską Wspólnotę Energii Atomowej</a></b></i> (<i>Traktaty rzymskie</i>) - dwie umowy międzynarodowe zawarte 25 marca <a href="/wiki/1957" title="1957">1957</a> przez sygnatariuszy Traktatu Paryskiego. Obie weszły w życie 1 stycznia <a href="/wiki/1958" title="1958">1958</a>. Dotyczyła powołania nowych organizacji - <a href="/wiki/Wsp%C3%B3lnota_Europejska" title="Wspólnota Europejska">Europejskiej Wspólnoty Gospodarczej</a> oraz <a href="/wiki/Europejska_Wsp%C3%B3lnota_Energii_Atomowej" title="Europejska Wspólnota Energii Atomowej">Europejskiej Wspólnoty Energii Atomowej</a>. Pierwsza zapoczątkowała integrację gospodarczą, druga - współpracę nad wykorzystywaniem <a href="/wiki/Energia_j%C4%85drowa" title="Energia jądrowa">energii jądrowej</a> w celach pokojowych.
